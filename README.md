# 权谋录 · 古代人生模拟器

一款以乾明王朝为背景的文字策略游戏。玩家扮演一名平民，在二十年间选择走上仕途、造反、经商或行侠，最终以自己的方式在乱世中留名。

**🎮 [立即游玩](https://hy031116.github.io/grow/)**

---

## 玩法简介

- **时间**：每回合代表一年，共二十年（乾明元年→二十年）
- **起点**：四种出身可选——寒门学子、边境武将、落魄商贾、游侠少年
- **赛道**：官场 / 造反 / 富商 / 侠客
- **目标**：在特定条件下触发胜利结局，或以资源积累水平决定"寿终正寝"的历史评价

## 结局体系

| 类型 | 数量 | 说明 |
|------|------|------|
| 胜利结局 | 14+ | 需在20年内满足特定条件（含高阶天命/盟主结局） |
| 死亡结局 | 12 | 第20年自然寿终，按资源分高中低三级 |
| 失败结局 | 若干 | 资源归零时触发 |

## 系统特性

- **NPC 关系系统**：4位关键人物，三层关系（疏远/中立/盟友），专属突破事件链
- **个人志向**：5种志向，影响行动加成与结局评价
- **手动战斗系统**：四赛道各2场战役，3轮策略选择，胜负影响资源走向
- **世界状态指标**：天下安定度 / 民间动荡，动态影响被动事件概率
- **人生时间线**：关键决策自动记录，结局页完整回顾一生
- **天下地图**：可交互 SVG 地图，展示势力范围
