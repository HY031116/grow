# 乾明廿载

一款以乾明王朝为背景的文字策略游戏。玩家扮演一名平民，在二十年间选择走上仕途、造反、经商或行侠，最终以自己的方式在乱世中留名。

**🎮 [立即游玩](https://你的用户名.github.io/grow/)**

---

## 玩法简介

- **时间**：每回合代表一年，共二十年（乾明元年→二十年）
- **起点**：四种出身可选——书生、武夫、商贾、游侠
- **赛道**：官场 / 造反 / 富商 / 侠客
- **目标**：在特定条件下触发胜利结局，或以资源积累水平决定"寿终正寝"的历史评价

## 结局体系

| 类型 | 数量 | 说明 |
|------|------|------|
| 胜利结局 | 12+ | 需在20年内满足特定条件 |
| 死亡结局 | 12 | 第20年自然寿终，按资源分高中低三级 |
| 失败结局 | 若干 | 资源归零时触发 |

## 系统特性

- NPC 关系系统（7位关键人物）
- 个人志向加成
- 世界状态指标（战乱度 / 皇权衰退等）
- 关键年份成长感悟（第5/10/15/20年）
- 危机事件系统（叛徒、政敌、追杀）

## 技术栈

纯 HTML / CSS / JavaScript，无依赖，零构建步骤。本地直接打开 `index.html` 即可运行。
