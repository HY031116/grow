# 月令系统设计 · 权谋录

> 设计版本 v0.3 — 从"行动点"到"四时辰制"
> 目标：让玩家体验"在古代规划一个月怎么过"的自由感

---

## 设计动机

旧版"每回合 3 个行动点"的核心问题：

| 问题 | 表现 |
|---|---|
| 选择同质化 | 第5回合和第15回合的操作逻辑完全一样 |
| 无时机概念 | "上奏折"在朝会和深夜做效果一样 |
| 被动事件砸脸 | 旱灾来了直接扣钱，玩家只能承受 |
| 缺乏规划感 | 3个点随便点，没有"这个月重点做什么"的策略 |

**月令制的核心思想**：把一回合从"3个同质行动点"变成"一个月内的4个不同时段"，不同时段做不同事的收益不同，玩家需要规划"这个月怎么安排"。

---

## 核心结构：一回合 = 一个月

```
【朔日朝会】辰时 · 朝堂
├── 政治/政务类行动效果 +30%
├── 上奏折、勤勉办差、效忠表态、举荐人才
└── 最适合：忠臣路线推进、争取圣眷

【旬中日常】午时 · 公务
├── 通用时段，无加成
├── 打探消息、查访、练兵、屯田、处理杂务
└── 杂事处理，适合不急的事务

【望日交际】申时 · 应酬
├── 社交/商贸/暗线类行动效果 +30%
├── 打点上司、结交外官、暗中交易、宴请
└── 最适合：扩大人脉、暗中布局

【月末私谋】戌时 · 暗室
├── 密谋/暗查/高风险类行动效果 +30%
├── 搜刮油水、暗中通敌、刺探、密会
└── 最适合：高风险高回报、不可告人之事

【天下有变】被动事件响应
├── 每月 65% 概率触发
├── 不占用主动时段，独立于四时辰之外
└── 玩家必须选择应对方式（不再自动承受）
```

## 时段加成机制

每个行动有 `preferredSlot`（偏好时段）和 `category`（行动类别）：

| 行动类别 | 偏好时段 | 加成效果 |
|---|---|---|
| political（政治） | 朔日朝会 | 效果 +30% |
| general（通用） | 旬中日常 | 无加成 |
| social（社交） | 望日交际 | 效果 +30% |
| intrigue（密谋） | 月末私谋 | 效果 +30% |

**跨时段执行**：行动可以在任意时段执行，但不在偏好时段时无加成。这迫使玩家做出选择——"等几天在朝会上奏" vs "现在就写但效果打折"。

## 大型行动（占2个连续时段）

某些关键行动需要投入更多时间，必须占满**两个连续时段**：

| 行动 | 所需时段 | 中断惩罚 |
|---|---|---|
| 挥兵攻城 | 旬中 + 望日 | 中途插入其他行动 → 攻城自动失败，兵力-15% |
| 大案钦查 | 朔日 + 旬中 | 中断 → 证据不足，无法结案 |
| 远征突袭 | 望日 + 月末 | 中断 → 行踪暴露，遭遇伏击 |
| 修建城防 | 任意连续2个 | 中断 → 工程烂尾，仅获50%效果 |
| 暗中通敌 | 月末 + 下月朔日 | 中断 → 密信被截，圣眷暴跌 |

**策略含义**：玩家要权衡"这个月全力做一件大事" vs "分散做四件小事"。大型行动的回报更高，但机会成本也更高——如果本月突然触发了一个需要应对的危机，你的大型行动可能就中断了。

## 连续策略：Buff 系统

连续在同一时段执行同类行动，会触发**累积加成**（Buff）：

| 连续次数 | Buff 名称 | 效果 |
|---|---|---|
| 连续3个月"朔日"做政治行动 | 朝堂重臣 | 政治类行动再+10%，皇帝更容易召见你 |
| 连续3个月"望日"做社交行动 | 八面玲珑 | 社交类行动再+10%，被动事件中有盟友帮忙的概率+20% |
| 连续3个月"月末"做密谋行动 | 暗流涌动 | 密谋类行动再+10%，但被发现的风险+15% |
| 连续2个月"告假归乡" | 韬光养晦 | 下月所有行动+30%（可叠加一次） |
| 连续3个月"屯田积粮" | 粮满仓 | 下3回合每回合钱粮+10 |

**策略含义**：玩家可以选择"专精一条路线"获得累积优势，也可以选择"灵活搭配"应对多变局势。专精有回报，但也有风险——如果你连续3个月都在搞密谋，可能会被人察觉。

## 被动事件改造：从"砸脸"到"选择"

旧版：被动事件自动执行，玩家只能承受。

新版：被动事件触发时，弹出**"天下有变"**响应窗口，玩家必须选择应对方式。

### 响应选项设计原则

每个被动事件提供 **3-4 个应对选项**：

1. **积极应对**：消耗当前资源或人脉，化危为机
2. **保守应对**：承受损失但保住核心计划
3. **冒险应对**：高风险高回报，可能触发更大危机或更大收益
4. **置之不理**（部分事件）：不消耗资源，但承受全额损失

### 示例

> **【天下有变】大旱之年，粮食减产，百姓流离**
>
> - **开仓赈灾**：下月钱粮-15，但民心+20，有概率触发"百姓感恩"后续事件
> - **勒紧腰带，军粮优先**：兵力稳定，民心-10，无额外损失
> - **趁机兼并土地**：钱粮+20，民心-25，有10%概率触发"民变危机"后续事件
> - **置之不理**：钱粮-18，民心-15，但本月计划不受影响

### 因果种子

每个应对选项不只是数值变化，还会**埋入因果种子**，影响未来事件：

- 选择"开仓赈灾" → 种子 `famine_relief` → 第8回合可能触发"百姓送万民伞"
- 选择"趁机兼并" → 种子 `land_grab` → 第10回合可能触发"民变告御状"
- 选择"置之不理" → 种子 `ignored_famine` → 第7回合可能触发"流民入匪"

## 时间叙事化

把"第X回合"变成**具体的年号+季节**，每个回合之间用过渡语桥接：

```js
const TIMELINE = [
  "建元三年 · 春",   // 第1回合
  "建元三年 · 夏",
  "建元三年 · 秋",
  "建元三年 · 冬",
  "建元四年 · 春",   // 第5回合
  "建元四年 · 夏",
  "建元四年 · 秋",
  "建元四年 · 冬",
  "建元五年 · 春",   // 第9回合
  // ...
];
```

### 过渡语示例

每回合结束时，根据当前局势生成一句过渡语：

- "冬去春来，又是一年。朝堂上的风向变了……"
- "盛夏炎炎，你的势力在这半年里悄然壮大……"
- "秋风肃杀，一场风暴正在酝酿……"
- "岁末天寒，这一年的因果，将在来年显现……"

## 状态机变化

```
[create] 选择性别 + 身份
    ↓
[route] 选择发展路线
    ↓
[play] 游戏主界面
    ├── 选择【朔日朝会】时段行动
    ├── 选择【旬中日常】时段行动
    ├── 选择【望日交际】时段行动
    ├── 选择【月末私谋】时段行动
    └── 点击【结束本月】
        ↓
        【天下有变】被动事件响应（如有）
        ↓
        月度结算：资源自然增减 + Buff 检查
        ↓
        主线剧情检查
        ↓
        胜负检查
        ↓
        round++ → 进入下月
```

## 与旧行动点制的对比

| 维度 | 旧制（3行动点） | 新制（四时辰） |
|---|---|---|
| 每回合可选行动 | 5种选3个 | 12种选4个 + 组合 |
| 行动同质性 | 完全相同 | 不同时段不同加成 |
| 被动事件 | 自动承受 | 玩家选择应对方式 |
| 策略深度 | 看数值补数值 | 规划时段 + 组合 + 连续策略 |
| 大型行动 | 无 | 有（占2时段，需权衡） |
| 时间感知 | "第X回合" | "建元X年·X季" |
| 人生感 | 弱（操作感） | 强（规划感+命运感） |

## 复杂度控制

时间槽制比行动点制复杂，MVP 阶段的控制策略：

1. **官场赛道先实现8种行动**，造反赛道先实现7种行动，留4-5种给后续版本。
2. **大型行动每种赛道只实现1个**（官场"大案钦查"、造反"挥兵攻城"）。
3. **Buff 系统先实现2个**（朝堂重臣 + 粮满仓），其余后续扩展。
4. **被动事件响应选项先实现2-3个**（积极/保守/冒险），不追求每个事件都有4个选项。
5. **时间叙事化先做年号+季节**，过渡语用固定模板池，不做动态生成。

## NPC 关系网（与月令制配合）

时间槽制为 NPC 关系网提供了天然的触发窗口——不同时段可能遇到不同的人：

| 时段 | 可能遇到的 NPC | 影响 |
|---|---|---|
| 朔日朝会 | 皇帝、宰相、御史 | 政治声誉变化 |
| 望日交际 | 张侍郎、外官、商人 | 人脉扩展或结怨 |
| 月末私谋 | 密探、刺客、暗线 | 高风险高回报的互动 |

NPC 的状态（好感度、记忆）会影响你在不同时段能做的事。例如：张侍郎对你好感度 < -20 时，你在"望日交际"中无法"结交外官"（被他暗中阻挠）。
