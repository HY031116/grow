# 核心玩法架构设计 · 权谋录

> 设计实现版 v1.3 — 古代中国人生模拟器

---

## 设计目标

> **让玩家穿越到古代，体验真实的第二人生。**
> 你可以是朝堂上翻云覆雨的权臣，也可以是招兵买马的乱世枭雄；
> 可以富可敌国操控天下财富，也可以仗剑走天涯行侠仗义。
> 没有唯一正确的人生——只有你亲手书写的命运。

---

## 核心主题

> **"穿越第二人生"** — 不是旁观历史，而是成为历史的一部分。

三层沉浸信息同时呈现：

```
时间层：乾明X年 · 季节（年龄称谓，X岁）   ← 我在哪个时代、哪个阶段
世界层：圣上勤政，百官各怀心思。            ← 世界正在发生什么
内心层：「寒窗苦读十数载，今日方踏入...」   ← 我的内心感受是什么
```

## 双层系统设计

### 层 1：经营模拟层（可见数值）

**资源体系（按赛道分组）：**

| 资源 | 主要赛道 | 说明 |
|---|---|---|
| 钱粮 | 全赛道 | 基础生存资源 |
| 兵力 | 诸侯/造反 | 军事基础 |
| 官位 | 忠臣/奸臣 | 朝堂地位（0-9品 → 宰相） |
| 圣眷 | 忠臣 | 皇帝信任度（影响仕途） |
| 权柄 | 奸臣 | 实际控制力（vs 圣眷对立） |
| 民心 | 造反/诸侯 | 百姓支持度（影响招兵和起事） |
| 财富 | 富商 | 独立财富积累（区别于钱粮） |
| 商路 | 富商 | 垄断的贸易线数量 |
| 武艺 | 侠客 | 个人战力 |
| 江湖名望 | 侠客 | 在江湖圈的影响力 |
| 恩义 | 侠客/全赛道 | 欠你人情的人数和质量 |
| 地盘 | 诸侯/造反 | 实际控制地域 |

> MVP 阶段简化：每条赛道只激活 **3-4 个核心资源**，其余共享"钱粮"作为通用资源。

**行动系统（月令 · 四时辰制）：**

一回合 = 一个月，每月包含 **4 个主动时段 + 1 个被动响应**：

```
【朔日朝会】辰时 · 朝堂
├── 政治/政务类行动效果 +30%
├── 上奏折、勤勉办差、效忠表态、举荐人才
└── 最适合：忠臣路线推进、争取圣眷

【旬中日常】午时 · 公务
├── 通用时段，无加成
├── 打探消息、查访、练兵、屯田、处理杂务
└── 杂事处理，适合不急的事务

【望日交际】申时 · 应酬
├── 社交/商贸/暗线类行动效果 +30%
├── 打点上司、结交外官、暗中交易、宴请
└── 最适合：扩大人脉、暗中布局

【月末私谋】戌时 · 暗室
├── 密谋/暗查/高风险类行动效果 +30%
├── 搜刮油水、暗中通敌、刺探、密会
└── 最适合：高风险高回报、不可告人之事

【天下有变】被动事件响应
├── 每月 65% 概率触发
├── 独立于四时辰之外，不占用主动时段
└── 玩家必须选择应对方式（不再自动承受）
```

**行动分类（按赛道解锁）：**

```
【官场行动】（忠臣/奸臣解锁，共 12 种）
├── 勤勉办差（political）—— +圣眷
├── 上奏折（political）—— 高风险高回报
├── 效忠表态（political）—— +圣眷，-权柄
├── 举荐人才（political）—— 消耗人脉换圣眷/权柄
├── 经营党羽（intrigue）—— +权柄，-圣眷
├── 打点上司（social）—— -钱粮，+圣眷/权柄
├── 结交外官（social）—— +人脉，可能触发联合事件
├── 暗中查访（intrigue）—— +情报，某些事件前提
├── 搜刮油水（intrigue）—— +钱粮，-圣眷
├── 养望乡里（general）—— +声望，长期收益
├── 告假归乡（general）—— 跳过2时段，下月+20%
└── 暗中通敌（intrigue）—— 占2时段，高风险

【造反行动】（诸侯/造反解锁，共 11 种）
├── 招兵买马（general）—— +兵力，-钱粮
├── 训练精兵（general）—— 兵力质量提升
├── 安民施粥（social）—— +民心，吸引投军
├── 挥兵攻城（general）—— 占2时段，大型行动
├── 屯田积粮（general）—— +钱粮，稳定后方
├── 刺探军情（intrigue）—— +情报，攻城前提
├── 修建城防（general）—— 占2时段，下月被动伤害-50%
├── 联合义军（social）—— +兵力，忠诚度存疑
├── 遣使谈判（political）—— 不战而屈人之兵
├── 偃旗息鼓（intrigue）—— 隐藏实力
└── 外出走动（general）—— 随机遭遇/人才/情报
```

**大型行动（占2个连续时段）：**

| 行动 | 所需时段 | 中断惩罚 |
|---|---|---|
| 挥兵攻城 | 旬中 + 望日 | 中途插入其他事 → 攻城失败，兵力-15% |
| 大案钦查 | 朔日 + 旬中 | 中断 → 证据不足，无法结案 |
| 暗中通敌 | 月末 + 下月朔日 | 中断 → 密信被截，圣眷暴跌 |
| 修建城防 | 任意连续2个 | 中断 → 工程烂尾，仅获50%效果 |

**连续策略（Buff 系统）：**

| 连续次数 | Buff 名称 | 效果 |
|---|---|---|
| 连续3个月朔日做政治 | 朝堂重臣 | 政治类+10%，皇帝召见概率+20% |
| 连续3个月望日做社交 | 八面玲珑 | 社交类+10%，盟友帮忙概率+20% |
| 连续3个月屯田积粮 | 粮满仓 | 下3回合每回合钱粮+10 |
| 连续2个月告假归乡 | 韬光养晦 | 下月所有行动+30% |

### 层 2：剧情叙事层（深度分支）

每个行动或被动事件触发一个**剧情节点**，包含：
- 场景描述（沉浸式第一人称 / 第三人称可选）
- 2-4 个选项，每个选项有明确后果预告
- 选择影响 Flag 状态（记录"你是什么样的人"）

**Flag 分类：**

```js
flags = {
  // 身份类
  "is_official": false,         // 是否入仕
  "is_rebel_leader": false,     // 是否宣布起义
  "is_merchant_guild_head": false, // 是否成为行会会首

  // 声誉类（连续值）
  "rep_loyal": 0,               // 忠义声誉 (0-10)
  "rep_ruthless": 0,            // 狠辣声誉 (0-10)
  "rep_benevolent": 0,          // 仁德声誉 (0-10)
  "rep_cunning": 0,             // 谋略声誉 (0-10)

  // 关系类
  "emperor_trust": 50,          // 皇帝信任度（仕途赛道核心）
  "enemy_list": [],             // 结怨势力列表
  "ally_list": [],              // 盟友势力列表

  // 剧情里程碑
  "survived_purge": false,      // 是否熬过了一次政治清洗
  "avenged_family": false,      // 是否替家人复仇
  "founded_kingdom": false,     // 是否建立了独立政权
}
```

## 核心循环（完整版）

```
【开局】
└─ 选择性别（影响部分事件文字）
└─ 选择出身（6种，决定初始资源和开放行动）
└─ 选择初始目标（自由选择赛道方向，或"随缘"）
    │
    ▼
【每回合：天下形势简报】
└─ 当前年号+季节（建元三年·春）
└─ 当前局势一句话描述
└─ 资源面板更新 + Buff 状态显示
    │
    ▼
【月令阶段】四时辰规划（玩家自主选择）
├── 【朔日朝会】选择政治/政务类行动
├── 【旬中日常】选择通用类行动
├── 【望日交际】选择社交/商贸类行动
├── 【月末私谋】选择密谋/暗查类行动
└── 大型行动需连续占满2个时段
    │
    ▼
【天下有变】被动事件响应（如有）
└─ 每月65%概率触发世界事件
└─ 玩家选择应对方式（积极应对/保守应对/冒险应对/置之不理）
└─ 应对方式埋入因果种子，影响未来事件
    │
    ▼
【月度结算】
└─ 资源自然增减（基于已建设产能、当前赛道、Buff状态）
└─ 大型行动完成结算（若未中断）
└─ 连续策略检查（是否触发Buff）
└─ Flag 检查是否触发里程碑事件
└─ 胜负检查 → 进入下月 或 触发结局
    │
    ▼
【时间过渡】
└─ 显示季节过渡语（"冬去春来，又是一年……"）
└─ round++ → 进入下月
```

## 赛道转换机制（人生转折）

这是"第二人生"感的关键设计——你不是被锁死在一条路上的。

| 转换场景 | 触发条件示例 | 后果 |
|---|---|---|
| 官员 → 造反 | 圣眷归零 + 兵力 > 20 | 失去所有官位，获得"乱贼"标签，开放造反行动 |
| 商人 → 诸侯 | 财富 > 80 + 民心 > 50 | 用财富购买军队，开始割据 |
| 侠客 → 忠臣 | 江湖名望 > 60 + 皇帝发出征召 | 接受朝廷招安，保留武艺但获得官职 |
| 造反 → 侠客 | 起义失败，只身逃脱 | 流落江湖，保留武艺，开放侠客行动 |
| 忠臣 → 奸臣 | 权柄 > 圣眷 × 2 + 政敌全部清除 | 可以选择"逼宫"剧情线 |

> **设计原则**：转换必须有代价，不能无成本切换。但转换本身不是惩罚，人生转折是游戏最戏剧性的时刻。

## 复杂度控制（防止范围膨胀）

这套设计最大风险是**内容爆炸**。控制策略：

1. MVP 只实现 **2 条赛道完整**（忠臣/奸臣路 + 造反路），其余赛道共享通用行动框架。
2. **剧情节点分层**：主线节点 10 个（精心分支）+ 通用节点 20 个（轻度分支）+ 随机事件池 15 个。
3. **Flag 在 MVP 阶段限制 12 个**，防止状态空间爆炸。
4. **赛道转换在 MVP 只实现 2 条转换路径**（官员→造反，侠客→忠臣）。
5. 每条赛道 MVP 激活 **3 个核心资源**，其余资源在后续版本扩展。
