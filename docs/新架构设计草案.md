# 新架构设计草案 · 权谋录 v2.0

> 目标：从"选择题游戏"升级为"真实第二人生模拟器"
> 核心问题：如何让玩家感受到"我的每一个选择都改变了这个世界"？

---

## 第一性原则分析

### 当前架构的根本问题

| 问题 | 本质原因 | 玩家感受 |
|---|---|---|
| 行动数量少 | 固定赛道锁定 5 种行动 | "我只能做这几件事" |
| 路线锁死 | 选完赛道 → 资源/行动全锁定 | "我没得选，只能走这条路" |
| 目标固定 | 胜利条件是系统硬写的 | "这不是我的人生，是系统给我的任务" |
| 世界无响应 | 选择结果只是数值变化 | "我做了什么好像没什么用" |

### 什么是"高自由度古代人生"的本质？

```
事实：人生的自由度来自于"时间 × 地点 × 人物关系 × 个人状态"的交叉组合
推导：游戏自由度 = 这四个维度的可选状态数量 × 各维度间的真实影响
结论：现有架构只有"个人状态"这一个维度，其余三个维度几乎没有
```

## 新架构三大核心模块

### 模块 1：开放行动池（Action Pool）

**推翻的旧设计**：选赛道 → 解锁对应的 5 种固定行动  
**新设计**：所有人共享基础行动池，额外行动由当前状态动态解锁

```
基础行动池（所有人都可用，约 8 种）
├── 游历四方 — 探索未知地点，随机遭遇（人物/机会/危险）
├── 读书增智 — 缓慢提升所有资源上限
├── 结交人脉 — 向某类 NPC 投入精力，提升关系
├── 休养生息 — 恢复钱粮，减少风险
├── 打探消息 — 获取情报，提前预知下回合事件
├── 接济百姓 — 积累民间声望（影响多种赛道）
├── 练习武艺 — 缓慢积累武艺值（所有人可用）
└── 写信联络 — 激活暂时中断的 NPC 关系

专属行动（基于当前身份/状态/关系动态解锁）
├── 在朝为官类：上奏折、打点上司、组建党羽（需要官职）
├── 拥兵割据类：招兵买马、挥兵攻城、安抚民心（需要兵力>10）
├── 经商致富类：开设商铺、打通商路、向皇商靠拢（需要财富>20）
├── 江湖行侠类：路见不平拔刀、闯荡门派、结义兄弟（需要武艺>10）
└── 跨界行动（关键设计）：
    ├── 官场人可以去江湖游历 → 获得江湖名声 → 解锁武林势力事件
    ├── 武将可以尝试经商 → 低效但可行
    └── 侠客可以接受朝廷招安 → 身份转变事件链
```

**核心机制**：行动池随游戏进程动态变化。玩家做了某件事 → 解锁新行动 → 世界感觉在扩展。

### 模块 2：个人志向系统（Personal Aspiration）

**推翻的旧设计**：圣眷≥85 或 权柄≥80 → 固定胜利  
**新设计**：玩家开局选择 1 个志向（或在游戏中途改变/发现）

```
可选志向（开局选择，影响终局评价维度）

权谋志向 — 你想成为历史上最有权势的人
  评价维度：圣眷 × 权柄（两者乘积）
  隐藏条件：flag[loyal] 和 flag[corrupt] 的比例决定"名臣"还是"权奸"

财富志向 — 你想富可敌国，掌控天下财货
  评价维度：财富 × 商路数量
  隐藏条件：flag[righteous] 决定"义商"还是"奸商"

武道志向 — 你想成为天下第一武者，名传江湖
  评价维度：武艺 × 名望
  隐藏条件：flag[sworn] 决定是独行侠还是一派宗主

霸业志向 — 你想推翻朝廷，建立自己的王朝
  评价维度：地盘 × 民心
  隐藏条件：flag[righteous] 决定"仁义之君"还是"铁血枭雄"

浪迹志向（反直觉选项）— 你什么都不想成为，只想活得自在
  评价维度：游历的地点数量 × 结交的 NPC 种类
  结局：不是"成功"也不是"失败"，而是独特的"自由人"结局
```

**关键设计**：志向不是终局胜利条件，而是评价视角。二十年结束时，游戏用你选择的维度来评价你的人生成就，但不会因为"没达标"而 game over。

### 模块 3：NPC 关系网络（NPC Network）

**推翻的旧设计**：NPC 只存在于剧情文本中，没有持续关系  
**新设计**：4-6 个固定 NPC，玩家可以主动投资关系

```
核心 NPC（按类型）

圣上 — 当朝皇帝（乾明帝）
  关系维度：圣眷值（已有）
  新增：皇帝有"性格弧线"，随游戏进程从"励精图治" → "晚年多疑"
  玩家动作：不直接与皇帝接触，通过行动改变圣眷

权相 — 朝中最有权势的大臣（初期是同盟候选，后期是竞争对手）
  关系维度：0-100 好感度
  玩家动作：结交（+好感）、竞争（-好感）、联合（临时结盟）
  影响：好感高 → 借助权相资源；好感低 → 触发政敌事件

江湖宗主 — 武林盟主（你的江湖身份的关键人物）
  关系维度：0-100 恩义值
  玩家动作：结义、相助、背叛
  影响：影响侠客路线所有事件

巨贾 — 天下首富（商业路线的核心 NPC）
  关系维度：0-100 生意关系
  玩家动作：合作、竞争、收购
  影响：影响富商路线所有事件

边关将领 — 镇守边关的将军（造反路线的关键）
  关系维度：0-100 军事盟约
  玩家动作：结盟、招降、对峙
  影响：影响造反/兵权路线

秘友 — 与玩家身份相近的知交好友（性别对应）
  关系维度：0-100 情谊值
  玩家动作：倾诉、相助、出卖
  影响：触发专属剧情链，情谊满格解锁隐藏事件
```

### 模块 4：世界状态响应系统（World State）

**推翻的旧设计**：世界状态只通过 WORLD_EVENTS 文字呈现，不影响玩法  
**新设计**：3个全局世界指标，玩家行为累计影响这些指标，指标反过来改变可选行动

```
朝廷稳定度（0-100，初始 60）
  ↑ 玩家多次效忠、帮助朝廷 → 升高 → 圣眷更容易获得
  ↓ 玩家多次腐败、造反 → 降低 → 圣眷事件频发、触发政变危机
  极低（<20）→ 触发"天下大乱"事件，所有赛道资源动荡

民间声望（0-100，初始 30）
  ↑ 接济百姓、行侠仗义、开仓放粮 → 升高 → 江湖/造反赛道buff
  ↓ 敛财苛民、暴力镇压 → 降低 → 触发民变事件
  极高（>80）→ 百姓自发相助，某些危机自动化解

江湖秩序（0-100，初始 50）
  ↑ 行侠仗义、维持公道 → 升高 → 侠客事件更多positive结果
  ↓ 黑道横行、武林动乱 → 降低 → 危机事件频率提升
  极高（>80）→ 玩家有机会成为武林盟主（不需要武艺极高）
```

| 内容 | 保留方式 |
|---|---|
| 所有结局文本（ENDINGS）| 直接复用，增加新维度结局 |

| 存档系统（localStorage）| 保留，更新 state 结构 |

## 实现阶段规划

### 阶段 1（MVP 重设，预计 3-4 轮）
- 实现动态行动池（基础8种+解锁机制）
- 实现志向选择界面
- 调整 state 结构以支持新模块

### 阶段 2（NPC 网络，预计 2-3 轮）
- 实现 NPC 关系面板（UI）
- 实现 NPC 关系影响行动解锁和事件结果
- 迁移现有故事事件为 NPC 关系事件链

### 阶段 3（世界状态，预计 2 轮）
- 实现三个全局世界指标
- 指标与行动/事件的双向影响
