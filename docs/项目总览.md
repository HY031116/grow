## Facts（已确认事实）

| 项 | 内容 |
|---|---|
| 类型 | 古代人生模拟器 / 文字策略 |
| 核心主题 | **"穿越第二人生"** — 让玩家体验古代的第二人生 |
| 背景 | 架空古代中国（乾明年号，共20年） |
| 目标平台 | Web（HTML/CSS/JS）零依赖，`file://` 直接运行 |
| 技术架构 | IIFE 模块模式 — `Game`（逻辑）/ `UI`（渲染）分离 |
| 状态机 | `create → track → ambition → play → story → transition → result` |
| 回合设计 | 每回合=1年，乾明元年→二十年，角色20-39岁 |
| 性别系统 | 男/女，女性有专属封号体系和性别化文本 |
| 存档 | localStorage 自动存档，支持继续游戏 |

## 系统架构速览（v1.8）

### 角色创建流程
```
选性别 → 选出身（4种）→ 选赛道（4种）→ 选志向（5种）→ 开始游戏
```

### 核心系统
| 系统 | 说明 |
|------|------|
| 赛道系统 | 4条赛道，各有独立资源/行动/胜负条件 |
| 志向系统 | 5种志向，影响通用行动的额外加成方向 |
| NPC关系 | 4个核心NPC，关系值0-100，三层事件，≥70解锁盟友结局 |
| 开放行动池 | 5个全赛道通用行动 |
| 世界状态 | 天下安定+民间动荡，动态影响事件概率和官场难度 |
| 事件系统 | 主线故事 + 危机 + NPC事件 + 被动事件（4层） |
| 结局系统 | 20+种结局变体，含NPC盟友专属结局×4 |
